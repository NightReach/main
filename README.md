# NightReach Ads Network (MVP)

Semi-grey Ad Network for gambling, crypto, adult, sweepstakes & push traffic.

MVP Features:
- Publisher onboarding
- Smartlink rotation system
- Offer management
- Click tracking
- Basic EPC optimization
- Affiliate-style advertiser setup

Tech:
- Next.js 14 (App Router)
- Prisma + PostgreSQL
- NextAuth
- Tailwind
